
# Overview

EasyMiner软件是采用参考引导的基因组装方式进行分子标记的提取。EasyMiner具有用户友好的Windows图形界面，可在个人电脑上高效准确地进行分子标记提取，无需依赖服务器。相较于其他软件，EasyMiner通过特别设计的拼接算法，能基于近源物种的参考基因组从二代测序中快速准确的提取分子标记，并同时兼具细胞器基因组组装、gb文件中基因序列分解、二代测序文件导出等功能。
